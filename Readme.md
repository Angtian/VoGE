Required Packages:
Numpy 1.20.3
PyTorch 1.10.2
PyTorch3D 0.6.1

Tested on:
Ubuntu LTS 16.04 with Nvidia RTX TITAN (CUDA 10.2)
Windows 10 with Nvidia RTX 3090 (CUDA 11.3)

cd demo

python LightDiffusion.py
